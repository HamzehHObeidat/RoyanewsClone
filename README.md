# Royanews Clone (React + Vite + Tailwind CSS)

A modern, RTL-ready news website clone built with **React**, **Vite**, and **Tailwind CSS**. This project replicates the structure and UI of a professional Arabic news portal, including hero sections, trending news, featured cards, category blocks, and a responsive mobile-friendly footer.
